# Arduino DHT11 BMP280 Environmental Monitoring Projects

## Objective

To develop and implement a collection of Arduino-based environmental monitoring projects that demonstrate temperature, humidity, atmospheric pressure, altitude measurement, weather monitoring, environmental dashboards, warning systems, and real-time data visualization.

This repository contains a collection of Arduino projects developed using DHT11 and BMP280 sensors.

---

## Projects Included

1. DHT11 Room Environment Monitor
2. Weather Monitoring System
3. Environmental Monitoring Dashboard
4. Environmental Warning System
5. Atmospheric Pressure and Altitude Monitor
6. Multi-Page Environmental Interface

---

## Components Used

### Hardware

* Arduino Uno
* DHT11 Temperature & Humidity Sensor
* BMP280 Pressure & Altitude Sensor
* SSD1306 OLED Display (128×64)
* LCD display(16x2)
* Breadboard
* Jumper Wires
* USB Cable

### Software

* Arduino IDE
* Wire Library
* Adafruit GFX Library
* Adafruit SSD1306 Library
* Adafruit BMP280 Library
* DHT Sensor Library

## Circuit Diagram

### OLED Connections

| OLED Pin | Arduino Uno |
| -------- | ----------- |
| VCC      | 5V          |
| GND      | GND         |
| SDA      | A4          |
| SCL      | A5          |

### BMP280 Connections

| BMP280 Pin | Arduino Uno |
| ---------- | ----------- |
| VCC        | 3.3V        |
| GND        | GND         |
| SDA        | A4          |
| SCL        | A5          |

### DHT11 Connections

| DHT11 Pin | Arduino Uno |
| --------- | ----------- |
| VCC       | 5V          |
| GND       | GND         |
| DATA      | D2          |


## Code Explanation

### Project 1 – DHT11 Room Environment Monitor

* Reads temperature and humidity using the DHT11 sensor.
* Displays real-time readings on the OLED display.
* Provides a simple room monitoring interface.

### Project 2 – Weather Monitoring System

* Measures temperature, humidity, pressure, and altitude.
* Displays weather-related environmental data.
* Updates readings continuously in real time.

### Project 3 – Environmental Monitoring Dashboard

* Combines multiple sensor readings on a single screen.
* Presents environmental information in dashboard format.
* Improves data visualization and monitoring.

### Project 4 – Environmental Warning System

* Continuously monitors environmental conditions.
* Compares readings against predefined thresholds.
* Displays warning messages when abnormal conditions occur.

### Project 5 – Atmospheric Pressure and Altitude Monitor

* Uses the BMP280 sensor to measure atmospheric pressure.
* Calculates altitude based on pressure data.
* Displays pressure and altitude values on the OLED.

### Project 6 – Multi-Page Environmental Interface

* Implements a multi-screen display system.
* Allows viewing different environmental parameters on separate pages.
* Demonstrates menu and page navigation concepts.
